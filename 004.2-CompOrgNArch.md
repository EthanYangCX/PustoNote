# CPU

常用的处理器架构有 ARM 和 x86。Intel 和 AMD 的 CPU 是基于 x86 架构，苹果的 m1 芯片就是基于 ARM 架构，高通也有很多基于 ARM。 x86 是复杂指令集 CISC，ARM 是精简指令集 RISC。
