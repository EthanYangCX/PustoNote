# WebGL

是一个 JS API，用于显示高性能的 2D 或 3D 图形。

WebGL本身相对比较繁琐，但一些优秀的js库简化了该流程，列如Three.js。

# WebAssembly (Wasm)

是一种语言以及对应的二进制格式，其性能方面优于 JS 而更接近更底层的语言，可以在主流的浏览器中运行，因此可以将其他语言写的性能更高的应用转换为 Wasm，以便在浏览器中调用，且快于 js 代码。
